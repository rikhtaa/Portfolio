# 🌐 Modern Full-Stack Portfolio

![Next.js](https://img.shields.io/badge/Next.js-14-black)
![Tailwind](https://img.shields.io/badge/TailwindCSS-3-blue)

![Portfolio Preview](./public/images/preview.png)

A high-performance portfolio built with **Next.js 14**, **Tailwind CSS**, and **Framer Motion**.  
Designed with a focus on clean aesthetics, scalable architecture, and seamless user interactions.

**[Live Demo](https://portfolio-seven-liart-61.vercel.app/)** | **[Report Bug](https://github.com/rikhtaa/Portfolio/issues)**

---

## ✨ Features

- **Interactive Visuals** – Immersive 3D elements built with Three.js and React Three Fiber  
- **Animated Components** – Dynamic background effects like ShootingStars and StarsBackground  
- **Project Showcase** – Apple Cards–inspired carousel for browsing projects  
- **Contact System** – Secure contact form powered by Next.js API routes and Nodemailer  
- **Responsive UI** – Mobile-first design optimized for all screen sizes  

---

## 🛠️ Tech Stack

- **Framework:** Next.js 14 (App Router)  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **3D Engine:** Three.js / React Three Fiber  
- **Backend:** Next.js API Routes & Nodemailer  

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/rikhtaa/Portfolio.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Configure environment variables

Create a `.env` file in the root directory and add your SMTP email credentials.

### 4. Run the development server

```bash
npm run dev
```
